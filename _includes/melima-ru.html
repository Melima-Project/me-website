Melima Core


Melima backend с использование LoopBack framework

Использование LoopBack позволяет создать быстрый и эффективный API и избежать проблем связанных с интеграцией с разными БД

Текущая версия melima core настроена для PostgresSQL, но благодаря LoopBack, вы можете подключить любую популярную БД

Melima Web


Melima Интерфейс Администратора (MAI)

MAI предназначен для создания новых продуктов, установки цен, количества вещей, для слежения за заказами, удаления заказов, применения скидок и много другого

MAI создан с использованием Bootstrap, jQuery, JsRender и jQuery Form Validator

Использование распространенных JavaScript библиотек позволяет MAI оставаться достаточно дружелюбным для начинающих разработчиков

Melima Droid


Melima Android Клиент

Главное приложение для использования конечным потребителем

Потребитель может ознакомиться с описанием товара, увидеть фотографию, сделать заказ или создать уникальный заказ

Используемые технологии: Retrofit и Gson для JSON сериализации в Java объекты, и их зависимости

В данный момент только начальный прототип с LoopBack аутентификацией доступен
